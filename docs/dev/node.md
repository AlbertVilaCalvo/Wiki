## REPL

Use `node` to start it. To exit, type `.exit`, or press Ctrl+C twice, or press Ctrl+D once.

Commands:

- `.help` - list commands
- `.break` or `.clear`
- `.editor`
- `.load <filename>` - load file
- `.save <filename>` - save commands you've run into a file
