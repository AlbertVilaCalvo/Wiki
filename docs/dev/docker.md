---
title: Docker
---

Start:
```bash
docker-compose -f docker-compose.yml up
```
Or:
```bash
docker-compose up -d
```

Connect to a container:
```bash
docker exec -ti sense_app bash
```

Shut down:
```bash
docker-compose down
```
